## 为什么选择 Debian？

在众多 Linux 发行版中，Debian 以其 **极致的稳定性**、**严格的自由软件原则** 和 **庞大的软件包仓库** 脱颖而出，被广泛用于服务器、桌面和嵌入式设备。

- **稳定性与可靠性**：Debian 的“稳定版”经过极其长时间的测试和修复，几乎不会出现崩溃或兼容性问题，这使得它成为生产环境中最受信赖的系统之一。
- **APT 包管理系统**：Debian 使用成熟的 APT（Advanced Package Tool）和 dpkg，依赖关系处理出色，软件安装、升级、卸载都极其方便，配合庞大的软件源（超过 5 万个软件包），几乎可以找到任何你需要的内容。
- **安全性与长期支持**：Debian 安全团队及时响应漏洞，稳定版会获得长达 5 年的安全更新支持（3 年完整支持 + 2 年 LTS），无需频繁重装系统。
- **纯粹的社区驱动**：Debian 完全由全球志愿者维护，不依赖于任何商业公司，确保了项目的开放性和中立性。
- **多架构支持**：支持 x86、ARM、PowerPC 等多种硬件平台，无论你在 PC、树莓派还是服务器上使用，都能找到对应版本。

对于任何希望获得一个 **稳定、安全、高效** 的操作系统的人来说，Debian 都是极具吸引力的选择。

## 为什么选择 Debian 12（Bookworm）而非 13（Trixie）？

Debian 的版本策略一向以稳定著称，其官方维护着三个主要分支：**稳定版（stable）**、**测试版（testing）** 和 **不稳定版（unstable）**。

- **Debian 12（Bookworm）**：当前的 **稳定版（stable）**，于 2023 年 6 月 10 日发布。它经过了严格的测试和审查，拥有长达 5 年的生命周期（3 年完整支持 + 2 年 LTS），是生产环境和日常使用的 **最优先推荐**。
- **Debian 13（Trixie）**：目前的 **测试版（testing）**，其发布日期 **尚未确定**。使用测试版意味着你会遇到更多频繁的软件包更新和潜在的不稳定因素。已有用户报告在安装或使用 Debian 13 时遇到了无法启动、无法登录等各类问题。

对于绝大多数用户，尤其是刚接触 Debian 的新手，选择成熟、稳定、有可靠支持周期的 **Debian 12（Bookworm）** 是更明智的选择。

## Debian 12（Bookworm）安装教程

### 第一步：准备工作

1. **下载 ISO 镜像**  
   访问 Debian 官方镜像站或国内镜像站（如清华大学开源软件镜像站），下载适合你电脑架构的 ISO 文件。  
   - 大多数现代个人电脑应选择 `amd64` 架构。  
   - 推荐下载 `DVD` 镜像（如 `debian-12.x.0-amd64-DVD-1.iso`），它包含了更多的软件包，可以减少安装后的下载量。

2. **制作安装介质**  
   使用 Rufus、Etcher 等工具，将下载好的 ISO 镜像刻录到一个 **至少 4GB** 的空白 U 盘中。

### 第二步：开始安装

1. **启动安装程序**  
   将制作好的 U 盘插入电脑，重启并在开机时选择从 U 盘启动。在出现的 Debian 安装菜单中，选择 **`Install`** 或 **`Graphical install`**（图形化安装）。

2. **语言和地区设置**  
   - **选择语言**：根据个人偏好选择，如 `中文（简体）` 或 `English`。  
   - **选择地区**：选择你所在的国家或地区，如 `中国` 或 `Hong Kong`。  
   - **选择键盘布局**：通常选择 `American English`。

3. **网络和账户配置**  
   - **配置主机名**：为你的电脑设定一个名称，例如 `debian-server`。  
   - **配置域名**：一般留空即可。  
   - **设置 Root 密码**：为超级管理员（root）设置一个强密码。**务必牢记！**  
   - **创建普通用户**：设置一个用于日常操作的非管理员用户名和密码。

4. **磁盘分区**  
   - **分区方法**：推荐新手选择 **`向导 - 使用整个磁盘`**。  
   - **选择磁盘**：选择你要安装 Debian 的硬盘。  
   - **分区方案**：推荐新手选择 **`将所有文件放在同一个分区中`**。如果想在重装系统时保留个人文件，可以选择 `单独设置 /home 分区`。  
   - **确认并写入磁盘**：在最终确认界面，选择 **`是`** 将分区方案写入磁盘。

5. **安装基本系统**  
   此时安装程序将开始复制文件，请耐心等待。

6. **配置软件包管理器（APT）**  
   - **扫描额外安装介质**：选择 **`否`**。  
   - **选择 Debian 镜像源**：**国家**选择 `中国`。**镜像站点**推荐选择国内的镜像源，例如 `mirrors.tuna.tsinghua.edu.cn`（清华大学）或 `mirror.lzu.edu.cn`（兰州大学），下载速度会快很多。  
   - **配置代理**：一般无需填写，直接选择 **`继续`**。

7. **软件包流行度调查**  
   选择 **`否`**。

8. **选择软件包**  
   这是选择安装桌面环境和系统服务的关键步骤。  
   - **如果你需要图形界面（桌面）**：可以勾选 `Debian desktop environment`，并选择一个桌面环境（如 `KDE Plasma` 或 `GNOME`）。  
   - **如果你需要远程管理**：**务必**勾选 `SSH Server`。  
   - **无论如何**，都建议勾选 `standard system utilities`（标准系统工具）。

9. **安装 GRUB 引导程序**  
   - 选择 **`是`** 将 GRUB 引导程序安装到主引导记录。  
   - 选择用于安装引导器的设备，通常是 `/dev/sda`。

10. **完成安装**  
    等待安装完成后，选择 **`继续`** 并重启电脑。重启后，移除 U 盘，系统将自动从硬盘启动。

### 第三步：安装后的基础配置（重要）

1. **登录系统**  
   使用你之前设置的 **普通用户** 账户和密码登录。

2. **更换软件源（换源）**  
   为了确保 `apt` 命令能快速下载和更新软件，需要将软件源配置文件 `sources.list` 中的地址改为国内镜像源。  
   - 打开终端（命令行）。  
   - 备份原文件：`sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak`  
   - 编辑文件：`sudo nano /etc/apt/sources.list`  
   - 删除或注释掉原有内容，粘贴上适合你地区的国内镜像源地址（例如清华源），然后按 `Ctrl+O` 保存，`Ctrl+X` 退出。

3. **更新系统**  
   在终端中执行以下命令，更新软件包列表并将所有已安装的软件包升级到最新版本：

   ```bash
   sudo apt update
   sudo apt upgrade
   ```
   
   至此，你已经成功在电脑上安装好了 Debian 12（Bookworm）操作系统。
